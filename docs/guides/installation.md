# Installation

## Core Library

To use QQuotes in your Node.js, Bun, or Deno project:

=== "npm"
    ```bash
    npm install @qquotes/core
    ```

=== "pnpm"
    ```bash
    pnpm add @qquotes/core
    ```

=== "bun"
    ```bash
    bun add @qquotes/core
    ```

## CLI Tool

To install the command-line interface globally:

=== "npm"
    ```bash
    npm install -g @qquotes/cli
    ```

=== "bun"
    ```bash
    bunx @qquotes/cli
    ```
